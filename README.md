# NLP Lab Programs

This repository contains Python programs developed for Natural Language Processing (NLP) lab practice.

## Programs Included

1. Reverse words in a sentence
2. Count vowels, consonants, digits and special characters
3. Word frequency counter
4. Remove stopwords using NLTK
5. Find palindrome words in a sentence
6. File statistics (count lines, words and characters)
7. Write user input to a file until "exit" is typed
8. Copy content from one file to another
9. Merge two text files

## Technologies Used

* Python
* NLTK Library
* Basic Text Processing
